# Collection of learning resource :memo:

[![pages-build-deployment](https://github.com/robertncl/Learning-resource/actions/workflows/pages/pages-build-deployment/badge.svg)](https://github.com/robertncl/Learning-resource/actions/workflows/pages/pages-build-deployment)

Curated, expert-reviewed books and courses (sourced from O'Reilly), each with a short summary. :white_check_mark:

With the following subjects:

* **AI** — foundations & math, machine/deep learning, generative AI, LLMs, and agents
* **DevOps** — culture, continuous delivery, SRE, observability & platform engineering
* **Cloud** — focused on **Azure** (plus AWS, Google Cloud, multicloud & FinOps)
* **Security** — focused on **application security** and **Azure cloud security**
* **Programming** — focused on **Python, JavaScript/TypeScript, Java & Go**
* **Linux/Unix** — command line, administration, systems programming & the kernel
* **Kubernetes** — core, patterns, networking & the CKA/CKAD/CKS certifications
* **Microservices** — patterns and distributed-systems foundations
* **Infrastructure as Code (IaC)** — Terraform, Ansible & Pulumi
* **GitHub** — Git, GitHub Actions/CI-CD, enterprise admin & **GitHub Copilot**
* **Tools & GUI** — curated CLI and GUI tool directories
